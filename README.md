This is a template for a crypto PWA that uses Accountkit.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Learn More

To learn more about AccountKit, take a look at the [AccountKit Documentation](https://accountkit.alchemy.com/).
